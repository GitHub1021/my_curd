### 简介 | Intro  

表格型内容管理系统，内置代码生成器，可用于业务逻辑快速开发。

### 优点 | Advantages  

- **轻量**：Jfinal、Easyui、Mysql5.7。
- **清晰**：系统分多个子模块，每个模块下分为model、controller、service 子包 和 相应页面模板。
- **快速**：前后端封装（弹窗、Tab切换、数据筛选过滤器、excel 导入导出、数据处理等）；页面级别权限管理、灵活的代码生成器。
