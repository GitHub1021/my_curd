my_curd 简介
========
基于 JFinal3.3  + Easyui1.5.3  权限系统。 <br>
页面级别权限，单表代码生成器 (快速开发)。

功能
--------
1. 角色、用户、菜单（权限）、组织机构 管理。<br>
2. 文件管理、文章管理。<br>
3. 代码生成器。 <br>
4. 网络小说下载 <br>
